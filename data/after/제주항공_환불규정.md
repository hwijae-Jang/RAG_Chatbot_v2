## 4. 제주항공 국제선 공항/고객센터 변경 시 추가 서비스 이용료

**키워드**: 서비스 이용료, 공항 카운터, 고객센터, 온라인 변경

제주항공 국제선 항공권을 온라인이 아닌 공항 카운터나 고객센터를 통해 변경하실 경우, 위의 변경 수수료에 추가로 서비스 이용료가 부과됩니다.

**적용일자**: 2024년 1월 2일부터 인상

| 변경 방법 | 서비스 이용료 |
|---------|-------------|
| 공항 카운터 / 고객센터 | 20,000원 / 20USD / 2,200JPY |
| 온라인 (홈페이지/모바일) | 무료 |

**예시**: 한국발 국제선 BASIC 운임을 공항 카운터에서 변경하면 변경 수수료 60,000원 + 서비스 이용료 20,000원 = 총 80,000원. 온라인 변경 시 60,000원만 부과.

## 5. 제주항공 국제선 유의사항

**키워드**: 유의사항, 환불 규정, 변경 규정, 24시간 면제, 노선 제한

- **구매처 확인**: 여행사 및 다른 항공사에서 구매한 경우 구매처로 문의
- **1인당 적용**: 수수료 및 위약금은 1인당, 구간당 각각 적용
- **게이트 노쇼 주의**: 출국장 입장 후 취소 시 게이트 노쇼 위약금이 별도 부과
- **환불 시점 기준**: 환불 시점을 기준으로 각 구간 당 시점별 위약금 적용
- **노선 제한**: 여정 변경은 동일 노선 내 날짜 변경만 가능하며, 노선 변경은 불가
- **운임 차액**: 변경 시 운임 차액과 수수료가 추가로 발생 가능
- **24시간 면제**: 최초 예약 후 24시간 이내 취소 시 수수료 면제 (단, 게이트 노쇼 제외, 제주항공 직접 구매 건만 해당)
- **소아 할인**: 국제선 소아 운임은 일부 운임에 한해 할인 혜택 적용
- **유아 면제**: 24개월 미만 유아는 수수료 면제

## 8. 제주항공 국내선 이름 철자 변경 수수료

**키워드**: 이름 변경, 철자 변경, 명의 변경, 수수료 면제

제주항공 국내선 항공권의 이름 철자를 변경하실 경우 수수료가 부과됩니다.

**적용일자**: 발권일 2022년 7월 20일부터 적용

### 8-1. 제주항공 이름 철자 변경 수수료

- **금액**: 5,000원 (성인/소아 동일)
- **조건**: 동일 발음 내 동일인 확인 후 철자 오류 시 성/이름 관계없이 변경 가능 (유효한 신분증 필요)
- **제한**: 타인으로의 명의 변경 불가
- **장소**: 공항 지점 또는 고객센터를 통해서만 가능
- **환불 불가**: 예약 취소 시에도 수수료는 환불되지 않음

### 8-2. 제주항공 이름 철자 변경 수수료 면제 조건

다음의 경우 제주항공 이름 철자 변경 수수료가 면제됩니다:

- 성/이름 역순 입력
- 성별 오입력
- 24개월 미만의 좌석 미점유 유아
- 개명으로 인한 이름 변경 시 증빙서류(개명 확인서 또는 초본) 제출
- 외국인의 경우 Middle Name 전체 추가

- **구매처 확인**: 여행사 및 다른 항공사에서 구매한 경우 구매처로 문의

- **환불 시점 기준**: 환불 시점을 기준으로 각 구간 당 시점별 위약금 적용

- **운임 차액**: 구매 시점과 다른 조건(날짜, 요일, 항공편 등)으로 변경 시 운임 차액 + 수수료/위약금 발생 가능
- **위약금 상한**: 지불한 운임보다 수수료가 큰 경우, 지불 운임만 위약금으로 적용
- **24시간 면제**: 최초 항공권 예약 후 24시간 이내 체크인 마감 전까지 취소 시 수수료 면제 (단, 제주항공 홈페이지/모바일/고객센터 구매 건만 해당)
- **재변경 수수료**: 24시간 이후에는 변경 시마다 수수료 부과
- **신분 할인**: 항공권 결제 완료 후에는 신분 할인 적용 불가, 재발권 필요
- **연령**: 성인·소아 동일 수수료 적용 / 유아(INF)는 면제
- **항공권 유효기간**: 미사용 시 발행일로부터 1년, 여행 개시 후 최초 출발일로부터 1년
- **환불 신청 기한**: 유효기간 만료일로부터 30일 이내
- **공항 이용료**: 미사용 항공권 시 전액 환불
- **재발권 수수료**: 변경 후 재발권된 항공권의 변경 수수료는 환불 불가

## 10. 제주항공 자주 묻는 질문 (FAQ)

**키워드**: FAQ, 자주 묻는 질문, 제주항공 변경, 제주항공 취소, 제주항공 환불

### Q1. 제주항공 국제선 BASIC 운임과 STANDARD 운임의 변경 수수료 차이는?

**A1**: 제주항공 한국 출발 국제선의 경우, BASIC 운임은 60,000원, STANDARD 운임은 40,000원으로 20,000원 차이가 납니다. 일본 출발은 BASIC 6,000엔, STANDARD 4,000엔으로 2,000엔 차이입니다.

### Q2. 제주항공 항공권을 공항에서 변경하면 추가 비용이 드나요?

**A2**: 네, 제주항공 국제선은 20,000원(또는 20USD/2,200JPY), 국내선은 10,000원의 서비스 이용료가 추가됩니다. 온라인으로 변경하시면 추가 비용이 없습니다.

### Q3. 제주항공 출발 91일 이전에 취소하면 수수료가 없나요?

**A3**: 제주항공 국제선 한국 출발의 경우, STANDARD와 BIZ LITE 운임은 출발 91일 이전 취소 시 위약금이 없습니다. 하지만 BASIC 운임은 60,000원의 위약금이 부과됩니다. 제주항공 국내선은 출발 61일 전까지 취소 시 BASIC 3,000원, STANDARD/FLEX 2,000원, BIZ LITE 1,000원의 수수료가 있습니다.

### Q4. 제주항공 게이트 노쇼 위약금이란 무엇인가요?

**A4**: 제주항공 게이트 노쇼 위약금은 탑승 수속 완료 후 탑승하지 않거나, 출국장 입장 후 취소하는 경우 부과되는 별도의 위약금입니다. 한국 출발 기준 200,000원이며, 일반 취소 위약금과 별도로 추가 부과됩니다.

### Q5. 제주항공 항공권 예약 후 24시간 이내 취소하면 수수료가 없나요?

**A5**: 네, 제주항공 홈페이지, 모바일 앱, 고객센터를 통해 직접 구매한 항공권은 예약 후 24시간 이내 체크인 마감 전까지 취소 시 수수료가 면제됩니다. 단, 게이트 노쇼의 경우는 제외되며, 여행사 등 제3자를 통해 구매한 항공권은 해당되지 않습니다.
