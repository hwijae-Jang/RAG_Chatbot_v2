# 개요
- 본 문서는 이스타항공 국내선/국제선의 **변경 수수료**, **취소 위약금(환불 수수료)**, **노쇼(NOSHOW) 수수료**, **채널별 서비스 규정**, **이름 변경 수수료**, **유의사항**을 체계화했습니다.
- 모든 금액은 **1인당·편도 기준**입니다. 통화는 표기된 단위를 따릅니다.
- 구매 후 **24시간 이내 변경/취소 수수료 면제** 규정이 있으며, 세부 조건은 유의사항을 참고하세요.

# 유의사항
- 본 이벤트/규정은 **결제일·탑승일 조건** 모두 충족 시 적용
- 조건 불일치 시 상위(기존) 수수료 규정 적용
- **홈페이지/모바일 앱 구매** 항공권에만 적용되는 조항 존재
- 국내선: 출발 전까지, 국제선: 출발 **1시간 전까지** 변경/취소 가능
- **구매 24시간 내** 변경/취소 시 수수료 면제
- 변경 시 **운임 차액 + 수수료** 발생 가능
- **NO-SHOW 수수료는 별도 부과**될 수 있음
- **성인/소아 동일**, **유아(INF) 면제**
- 항공권 **유효기간**: 미사용 1년(발행일 기준) / 일부 사용 1년(최초 출발일 기준)
- 세금은 전액 환불 가능
- 출발편 미탑승 시 **귀국편 자동 취소 + 수수료 부과** 가능성

# FAQ 요약
- **국제선 날짜 변경**: 특가운임 기준 출발 14일 전까지 무료(무기한 연장 공지 포함)
- **국제선 노쇼**: 출발 1시간 전 이내 또는 출발 이후 취소 시 고액 위약금 또는 운임 100% 적용 가능
- **국내선 노쇼**: 특가운임 운임 100% 징수, 할인/일반 운임 20,000원
